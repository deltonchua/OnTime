What is MetaMask?

MetaMask is a web extension, which allows you to manage your Ethereum private keys via your web browser.


What Can I Do With MetaMask?

Source: MetaMask - Chrome Web Store


MetaMask serves as a wallet for Ether and ERC-20 tokens in your browser.


Add OTO to your MetaMask wallet:


It allows access to Ethereum-enabled distributed applications, or "DApps". By injecting the Ethereum web3 API into every website's javascript context, MetaMask allows DApps to read from the blockchain.


Besides, MetaMask also lets the user create and manage their own identities. This can be done via private keys, local client wallet and hardware wallets like Ledger or Trezor. When a DApp wants to perform a transaction and write to the blockchain, the user gets a secure interface to review the transaction, before approving or rejecting it.

How Do I Install MetaMask?

Visit https://metamask.io/ and install the MetaMask extension for your browser (Chrome, Firefox or Brave).
